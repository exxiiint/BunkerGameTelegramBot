# Bunker Game Telegram Bot
Hello!
Telegram-бот, который переносит карточную игру «Бункер» из реальной жизни в онлайн-формат.
Игроки создают комнаты, получают случайные роли и проходят обсуждения и голосования прямо в боте.
Сервис автоматизирует геймплей, хранит историю партий и делает игру доступной в любое время через Telegram.
